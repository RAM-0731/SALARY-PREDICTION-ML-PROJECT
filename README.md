# SALARY-PREDICTION-ML-PROJECT
This project is a simple Machine Learning web app built using Streamlit.
It predicts salary based on years of experience and skill level.
The model used is Linear Regression from Scikit-learn.
Users can input values using interactive sliders.
The app instantly shows predicted salary results.
It also displays a salary range for better understanding.
A dataset is used to train the model for prediction.
Basic data visualization is included for insights.
This project is useful for beginners learning ML concepts.
It can be further improved using real-world datasets and advanced models.
